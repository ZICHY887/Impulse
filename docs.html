SYSTEM DOCUMENTATION

Operational Manual for AEPNKJ IMPULSE V3.0. Read before deployment.

01. INSTALLATION PROTOCOL

Impulse V3 is an unsigned extension. You must install it manually into the Adobe CEP directory.

Windows Path:

C:\Program Files (x86)\Common Files\Adobe\CEP\extensions\

Mac OS Path:

/Library/Application Support/Adobe/CEP/extensions/

1. Extract the downloaded folder.
2. Move the entire com.aepnkj.impulse folder into the path above.
3. Restart Adobe After Effects.

4. Right Click > New > String Value
5. Name: PlayerDebugMode | Value: 1

03. INTERFACE OVERVIEW

The panel is divided into three strategic sectors:

  • MAIN: Core animation tools (Ease, Anchor, Wiggle).
  • TOOLS: Layer manipulation (Cuts, Duration, Swapper).
  • SYS: Settings, Visual Themes, and Cache Nuke.

05. WORKFLOW SHORTCUTS

Impulse supports modifier keys for advanced functionality:

  • Smart Pre-Comp (Click): Pre-composes each selected layer individually.
  • Smart Pre-Comp (Shift + Click): Merges all selected layers into ONE pre-comp.
  • Trim In/Out: These buttons respect the Playhead (CTI) location accurately.
